What is the formula in finding the n?

(A) n = t m

(B) n = ( F P ) 1 n − 1

(C) ** n = log ⁡ ( F P ) log ⁡ ( 1 + j )

(D) None of these

EXPLANATIONS BELOW

Concept note-1: -Step 1: The nth term of an arithmetic sequence is given by an = a + (n – 1)d. So, to find the nth term, substitute the given values a = 2 and d = 3 into the formula. Step 2: Now, to find the fifth term, substitute n = 5 into the equation for the nth term.

Concept note-2: -logarithm, the exponent or power to which a base must be raised to yield a given number. Expressed mathematically, x is the logarithm of n to the base b if bx = n, in which case one writes x = logb n. For example, 23 = 8; therefore, 3 is the logarithm of 8 to base 2, or 3 = log2 8.

Concept note-3: -log a = x means that 10x = a . 10log a = a (The basic logarithmic identity). log(a/b) = log a-log b, a > 0, b > 0. log an = n (log a) (Logarithm of a power).
